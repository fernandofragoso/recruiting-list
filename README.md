# recruitinglist
>React / ES6 / HTML / CSS

### Instructions

1. Clone this repository.

2. Install dependencies:

```shell
yarn
```
or
```shell
npm install
```

3. Run dev server:

```shell
yarn start
```
or
```shell
npm start
```

### Tests

1. Run tests

```shell
yarn test
```
or
```shell
npm test
```
